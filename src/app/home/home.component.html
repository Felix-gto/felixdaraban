Despre mine

About me

QA tester cu o pasiune pentru frontend development.

QA tester with a passion for frontend development.

Am descoperit IT-ul în iunie 2021 când mi s-a propus poziția de software QA tester.

Am profitat de oportunitate și am decis să invaț mai multe, începând cu HTML, CSS, JavaScript și continuând cu Angular ca și framework de frontend. Cu cât descopeream mai multe lucuri noi, cu atât deveneam mai curios și determinat să evoluez.

Îmi place să explorez o aplicație, să-i studiez funcționalitățile descoperind unde sunt probleme, cum poate fi îmbunătățită și să lucrez împreună cu developerii pe întreg parcursul dezvoltării aplicației.

I discovered IT in June 2021 when I was given the position of software QA tester.

I took the opportunity to go deeper and learn the basics of programming, started studying HTML, CSS and JavaScript and continued with the frontend framework Angular. The more I learned the more curious and excited I became, determined to evolve.

I love exploring an application, studying it's features, discovering how it can help and how can it be improved, working together with developers to help them throughout the software development lifecycle.
